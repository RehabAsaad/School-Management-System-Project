Description:

School Management System is a simple Java console application that helps manage student records and course enrollments. It allows adding students, enrolling them in courses, removing courses, and viewing student information.

Features:

1-Add New Student: Register a student with an ID and password.
2-Enroll in a Course: Enroll students in one of the available 6 courses.
3-Remove a Course: Unenroll a student from a course.
4-View Student Information: Display student details and enrolled courses.
5-Interactive Menu: Easy navigation through the system with a text-based menu.

Technologies Used:

1-Java
2-OOP (Object-Oriented Programming)
3-Arrays for storing students and courses.
4-Scanner for user input.
